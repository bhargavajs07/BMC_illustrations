Katapayadi Sankhya Encoding for Melakartha Ragas

Katapayadi Sankhya System

This ancient Indian system assigns numerical values to consonants. Note that न (na) = 0 in the traditional system:

Consonant Mapping:

  • Ka: क=1, ख=2, ग=3, घ=4, ङ=5
  • Cha: च=6, छ=7, ज=8, झ=9, ञ=0
  • Ta1: ट=1, ठ=2, ड=3, ढ=4, ण=5
  • Ta2: त=6, थ=7, द=8, ध=9, न=0
  • Pa: प=1, फ=2, ब=3, भ=4, म=5
  • Ya: य=1, र=2, ल=3, व=4, श=5, ष=6, स=7, ह=8

Examples:

  • कनकांगि → क=1, न=0 → 10 → 01
  • रत्नांगि → र=2, त=6 → 26 → 62
  • गणमूर्ति → ग=3, न=0 → 30 → 03

For Melakartha ragas, the first two consonants are encoded, then the digits are reversed to get the raga number (1-72).

Katapayadi Sankhya Encoding for Melakartha Ragas

Katapayadi Sankhya System

This ancient Indian system assigns numerical values to consonants:

  • Ka-group (क, ख, ग, घ, ङ) → 1
  • Cha-group (च, छ, ज, झ, ञ) → 2
  • Ta-group (ट, ठ, ड, ढ, ण) → 3
  • Ta-group (त, थ, द, ध, न) → 4
  • Pa-group (प, फ, ब, भ, म) → 5
  • Ya-group (य, र, ल, व, श, ष, स, ह) → 6, 7, 8, 9, 1, 2, 3, 0

For Melakartha ragas, the first two consonants are encoded, then the digits are reversed to get the raga number (1-72).

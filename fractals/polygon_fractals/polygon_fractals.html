How it Works:

Interpolation Parameter (p): Controls how far along each edge the new polygon's corners are placed. p=0.1 means corners are placed 10% along each edge.

Number of Shapes: Total number of nested polygons to generate. Each new polygon is created from the corners of the previous polygon.

Rotations per Shape: Number of 360Â° rotations performed after each polygon is added (rotation speed stays constant).

Base Shape: The initial regular polygon used as the starting shape for the fractal generation.

Animation Sequence: Corner marking (0.4s each), edge drawing (1s), then rotations (4s total) before adding the next polygon.
